# Jarvis Capabilities Reference

## Core Operations
- **Terminal**: Full Linux shell access
- **File Management**: Read, write, search, patch files
- **Code Execution**: Python with tool access
- **GitHub**: Full repo management via gh CLI

## Research
- **Web Search**: General internet search
- **X Search**: Twitter/X post search
- **arXiv**: Academic paper search
- **Session Search**: Search past conversations

## Media
- **Image Generation**: Create images from text
- **Video Analysis**: Analyze video content
- **Video Generation**: Create videos
- **Text-to-Speech**: Convert text to audio

## Communication
- **WhatsApp**: Primary platform (no markdown)
- **Send Messages**: To connected platforms
- **Clarify**: Ask user questions

## Automation
- **Cron Jobs**: Schedule recurring tasks
- **Delegate Tasks**: Spawn sub-agents
- **Memory**: Persistent storage
- **TODO Lists**: Task management

## Skills System
- Load skills with `skill_view`
- Create skills with `skill_manage`
- Skills stored in ~/.hermes/skills/
- Also backed up to jarvis/memory/skills/
