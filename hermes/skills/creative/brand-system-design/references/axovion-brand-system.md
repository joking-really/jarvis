# Axovion Brand System Example

This is a reference example of a complete brand system document for an AI automation services startup (Axovion). It demonstrates the full structure and depth defined in the `brand-system-design` skill.

**Brand:** Axovion  
**Industry:** AI automation services (B2B, ecommerce, SaaS)  
**Aesthetic:** Cyber-premium, dark-mode, AI-native startup  
**Reference brands:** Linear, Vercel, Vapi, ElevenLabs

## 2. Target Audience

| Attribute | Detail |

### Industries

- Ecommerce (DTC brands, Shopify stores)
- SaaS & B2B services
- Coaching, consulting, and agencies
- Real estate and property
- Professional services (legal, financial, medical)
- High-ticket sales organizations

| Pain Point | Underlying Fear |
|------------|-----------------|

### Buying Psychology

| Trigger | How to Activate |
|---------|---------------|
| **FOMO on AI** | Show what competitors are already doing |
| **Loss aversion** | Quantify leaked revenue from slow follow-up |
| **Status** | Position AI adoption as a leadership move |
| **Simplicity** | Remove complexity; sell the outcome, not the mechanism |

- Relief ("Finally, this is handled")
- Excitement ("This scales everything")

- Being sold to by marketers who have never built a system

- Long implementation timelines
- Systems that require constant babysitting

## 3. Brand Voice

### Tone

| Context | Tone |
|---------|------|
| **Primary** | Confident, sharp, direct, slightly irreverent |
| **Educational** | Clear, authoritative, no fluff |
| **Promotional** | Urgent but never desperate. Premium but accessible. |
| **Social** | Conversational, contrarian when appropriate, meme-literate |

Axovion does not sound like a corporation. Axovion sounds like the smartest person in the room who actually ships.

### Writing Style

- Short sentences. Punchy rhythm.
- No filler adjectives ("revolutionary," "cutting-edge," "innovative").
- Use specifics over superlatives.
- Active voice always.
- Numbers front-loaded when possible.

**Good:** "We built an AI agent that booked 47 meetings in 10 days. No human touched a lead."
**Bad:** "Our innovative AI solution revolutionizes the way businesses engage with potential customers."

### Messaging Style

| Principle | Application |
|-----------|-------------|
| **Lead with outcome** | Never lead with feature or technology |
| **Quantify everything** | "$43K recovered" beats "improved revenue" |
| **Name the enemy** | Manual follow-up. Slow response times. Leaked leads. |
| **Own the contrarian take** | "Chatbots are dead. AI agents are the future." |
| **Speak the customer's language** | Use their terminology, not AI jargon |

### CTA Style

| Type | Example |
|------|---------|
| **Direct** | "Book your AI audit" |
| **Curiosity** | "See what your follow-up is actually costing you" |
| **Exclusivity** | "We build 4 systems per month. Claim your slot." |
| **Risk reversal** | "Free revenue leak analysis. No pitch unless you want one." |

Never use: "Learn more," "Click here," "Discover how," "Get started today" (generic).

### Sales Tone

- Consultative but decisive.
- Diagnose before prescribing.
- Challenge the client's assumptions.
- Frame Axovion as a strategic partner, not a vendor.

**Good:** "Your current follow-up sequence is losing 30% of warm leads. Here's the data."
**Bad:** "We have a great product that can help with your lead management needs."

### Words Frequently Used

autonomous, deploy, system, revenue, convert, scale, agent, pipeline, qualify, book, nurture, leak, recover, execute, integrate, stack, workforce, 24/7, done-for-you, audit, build, ship, results

### Words to Avoid

| Avoid | Replace With |
|-------|--------------|
| chatbot | AI agent, revenue agent |
| software | system, engine |
| solution | system, approach |
| innovative | (delete) |
| cutting-edge | (delete) |
| revolutionary | (delete) |
| leverage | use |
| synergize | (delete) |
| streamline | fix, automate |
| optimize | improve, fix |
| best-in-class | (delete) |
| world-class | (delete) |

### UI Style

- Dark-first design. All default surfaces are `#0A0A0F` or `#141419`.
- Glassmorphism for overlays: `background: rgba(20, 20, 25, 0.8); backdrop-filter: blur(12px);`
- Subtle grid patterns or dot matrices as background texture (opacity 0.03–0.05).
- Cards have 1px border `#1E1E24` with no border or subtle gradient border on hover.
- No heavy drop shadows. Use glow instead.

Avoid traditional shadows. Use glow effects:

| Name | Value | Usage |
|------|-------|-------|
| **Blue Glow** | `0 0 20px rgba(0, 102, 255, 0.3)` | Primary CTAs, active states |
| **Cyan Glow** | `0 0 30px rgba(0, 240, 255, 0.2)` | Data highlights, metrics |
| **Purple Glow** | `0 0 25px rgba(123, 97, 255, 0.25)` | Premium elements |
| **Inner Glow** | `inset 0 1px 1px rgba(255, 255, 255, 0.05)` | Subtle depth on inputs |

### Iconography Style

- Line icons, 1.5px stroke, rounded caps.
- Icon library: Phosphor Icons (duotone or regular weight).
- Icons in `#8A8F98` or `#FFFFFF`. Active states in `#0066FF` or `#00F0FF`.
- No filled icons except for status indicators.
- Animated icons: subtle pulse or glow on active states.

### Animation Style

| Type | Specification |
|------|---------------|
| **Easing** | `cubic-bezier(0.16, 1, 0.3, 1)` (expo out) for entrances |
| **Duration** | 200ms for micro-interactions, 400ms for reveals, 800ms for page transitions |
| **Entrance** | Fade up 16px + opacity 0→1 |
| **Hover** | Scale 1.02, glow intensify, 150ms |
| **Loading** | Pulsing gradient or orbital dots |
| **Scroll** | Parallax depth layers, staggered reveals |

## 5. Social Media Design Direction

### Post Styles

| Format | Specification |
|--------|---------------|
| **Single Image** | Dark background, large typography, single metric or statement. Minimal elements. |
| **Video/Reel** | Fast cuts (1–2s), text overlays, dark aesthetic, electronic or ambient audio. No stock footage of people shaking hands. |
| **Carousel** | 5–10 slides. Cover = hook. Body = proof/education. Final = CTA. Consistent dark template. |
| **Meme/Relatable** | Can break format for engagement, but must use brand colors and typography. |

### Carousel Styles

**Template Structure:**
1. **Cover:** Bold statement or question. Large text. No fine print.
2. **Problem:** Agitate the pain. Use their words.
3. **Mechanism:** How it works (simplified, visual).
4. **Proof:** Number, screenshot, or testimonial.
5. **Objection Handler:** Address the "yeah but..."
6. **CTA:** Direct next step. QR code or link in bio reference.

**Design Rules:**
- Each slide must work standalone (people scroll out of order).
- Text max 12 words per slide for readability.
- Use `#00F0FF` for data points and metrics.
- Progress indicator: small dots or line at bottom.

### Ad Styles

| Element | Rule |
|---------|------|
| **Headline** | 5 words or less. Outcome-focused. |
| **Visual** | Dark UI mockup, dashboard, or abstract AI visualization. No generic business stock. |
| **Body** | One sentence. One metric. One promise. |
| **CTA Button** | High contrast. Blue glow. Action verb. |
| **Social Proof** | Badge or micro-testimonial below CTA. |

### Thumbnail Styles

| Platform | Specification |
|----------|---------------|
| **YouTube** | Dark background, 3-word title, face optional (high contrast if used), electric cyan or blue accent element, 1280x720 |
| **LinkedIn** | Clean, text-forward, single bold statement, brand colors, 1200x627 |
| **Instagram** | Square or 4:5, bold typography, minimal, high contrast |

**Thumbnail Rules:**
- Text must be readable at 200px width.
- One focal point only. No clutter.
- Use glow effects, not drop shadows.
- Faces: if used, high contrast, looking toward text or CTA.

### Layout Preferences

- Asymmetric balance. Weight on left or center. Never centered symmetry.
- Generous negative space. Content breathes.
- Z-pattern or F-pattern reading flow.
- Visual anchors: large numbers, bold statements, or glowing elements.

### Visual Hierarchy

1. **Primary:** The one thing you want them to remember (largest, brightest)
2. **Secondary:** Supporting proof or context (medium, muted)
3. **Tertiary:** CTA or attribution (small, precise)

### Text Placement Rules

- Safe zone: 60px from edges for all text.
- Max 3 text elements per visual.
- Text alignment: left or center. Never justified.
- Line length: max 8 words per line for headlines.

### CTA Placement

- Bottom right for static images.
- Center-bottom for carousels (final slide).
- Embedded in video as end-frame (last 3 seconds).
- Always visible without scrolling or clicking "more."

### Branding Placement

- Subtle logo mark: bottom right corner, 24px height, `#8A8F98` or white, 40% opacity.
- Full logo: only on final slides or landing pages.
- Brand name in caption, not always on visual.
- URL: axovion.io — always lowercase, always included in bio-linked content.

## 7. Creative Psychology Rules

### What Makes Content Attention-Grabbing

| Principle | Application |
|-----------|-------------|
| **Pattern interrupt** | Start with what they don't expect: a number, a confession, a contradiction |
| **Specificity** | "$43,217" beats "increased revenue" |
| **Visual contrast** | Dark + neon. Minimal + bold. Empty + one focal point |
| **Movement** | In video: fast cuts, text animations, progress indicators |
| **Familiar + surprising** | Use their language, then subvert their expectation |

### Hook Strategies

| Hook Type | Formula | Example |
|-----------|---------|---------|
| **The Confession** | "I used to think [common belief]. Then I [discovered truth]." | "I used to think chatbots were the future. Then I built an AI agent." |
| **The Number** | "[Specific number] [outcome] in [timeframe]. Here's how." | "47 meetings booked in 10 days. No human touched a lead." |
| **The Contrarian** | "[Common advice] is wrong. Here's what actually works." | "Follow-up speed doesn't matter. Response quality does." |
| **The Question** | "What if [painful scenario] could be [desirable outcome]?" | "What if every lead got followed up in under 60 seconds?" |
| **The Roast** | "[Common mistake] is costing you [specific loss]." | "Your 'we'll get back to you' auto-reply is burning $12K/month." |
| **The Before/After** | "Before: [pain]. After: [result]. The difference: [mechanism]." | "Before: 3-hour response time. After: 8 seconds. The difference: one AI agent." |

### Curiosity Patterns

- Open loops: "There's one setting that doubled our conversion rate..."
- Withheld information: "The reason most AI projects fail has nothing to do with the AI."
- Sequences: "Step 3 is where everyone messes up."
- Counterintuitive claims: "Slowing down your follow-up can increase conversions by 40%."

### Conversion-Focused Messaging

| Principle | Application |
|-----------|-------------|
| **One job per message** | Each post, slide, or ad makes ONE point |
| **Quantify the cost of inaction** | "Every day you wait, [X] leads go cold" |
| **Remove ambiguity** | "Book a 15-minute audit" not "Let's talk" |
| **Stack proof** | Number + name + screenshot + timeframe |
| **Make it feel inevitable** | "This is where every business is headed. The question is when." |

### Authority-Building Techniques

- Show the work: screenshots, architecture, actual builds
- Name specific tools and integrations (not generic "AI")
- Share failures and pivots, not just wins
- Use technical depth sparingly but precisely
- Quote clients with full names and company names when possible

### Trust-Building Patterns

| Pattern | Example |
|---------|---------|
| **Specificity** | "$43,217 in recovered revenue" not "significant ROI" |
| **Vulnerability** | "We messed up the first deployment. Here's what we learned." |
| **Transparency** | Show pricing ranges, timelines, and requirements upfront |
| **Social proof density** | Multiple testimonials, not one perfect one |
| **Risk reversal** | "Free audit. No pitch unless you want one." |

## 9. AI Generation Instructions

### How Visuals Should Look

- **Default mood:** Dark, premium, technical-but-accessible.
- **Background:** `#0A0A0F` or subtle gradient to `#141419`. No pure black.
- **Text:** White or light gray. High contrast always.
- **Accents:** Electric cyan (`#00F0FF`) for data and metrics. Axovion blue (`#0066FF`) for CTAs and primary actions.
- **Imagery:** Abstract AI visualizations, network nodes, flowing data, dashboard UIs, minimal geometric shapes. No stock photos of people in meetings.
- **Depth:** Use subtle glows, glass layers, and grid textures. No heavy shadows or 3D bevels.
- **Density:** Minimal. One focal point. Generous negative space.

### What Aesthetic to Follow

| Attribute | Direction |
|-----------|-----------|
| **Genre** | Cyber-premium, dark-mode SaaS, AI-native |
| **Reference brands** | Vercel, Linear, Vapi, ElevenLabs, Replicate |
| **Mood** | Confident, precise, slightly futuristic, never corny |
| **Texture** | Smooth, clean, occasional subtle grid or dot pattern |
| **Motion** | Fast, precise, exponential easing, glow pulses |

### What Emotional Response Content Should Create

| Stage | Emotion |
|-------|---------|
| **First impression** | Intrigue. "This looks different." |
| **Engagement** | Recognition. "They understand my problem." |
| **Consideration** | Aspiration. "This is where I need to be." |
| **Conversion** | Confidence. "These people can actually build this." |

### What Style Should NEVER Be Generated

| Avoid | Why |
|-------|-----|
| **Corporate stock photography** | Kills premium positioning instantly |
| **Bright, colorful backgrounds** | Off-brand, feels cheap |
| **3D cartoon characters or mascots** | Undermines authority |
| **Gradient text on gradient backgrounds** | Illegible, amateur |
| **Drop shadows on everything** | Dated, heavy |
| **Generic "tech" imagery** (circuits, globes, handshakes) | Cliché, meaningless |
| **Overcrowded layouts** | Dilutes message, reduces conversion |
| **Exclamation marks in headlines** | Desperate, not confident |
| **Rainbow color schemes** | Unfocused, unprofessional |
| **Comic Sans or playful fonts** | Destroys credibility |

### What Makes Content "On-Brand"

Checklist for every asset:
